# configs
some configuration files
## Installation
run:

    install.sh args

args can be any combinations from "all","bash", "gdb", "vim"

## crontab
    crontab ./crontab.txt

## tmux
    cp tmux/.tmux.conf ~/

## ssh
    /etc/ssh/ssh_config
    ChallengeResponseAuthentication no
    PasswordAuthentication no
